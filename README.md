# Question Paper Generation System

## Overview
This project is designed to simplify and automate the question paper generation process for universities. It reduces manual effort by generating a structured and formatted question paper from existing question bank PDFs.

## Problem Statement
Universities often maintain large question banks in PDF format. Creating question papers manually from these files is time-consuming, error-prone, and repetitive. This project aims to streamline the process by automating question extraction and paper structuring.

## Features
- Accepts question banks in PDF format  
- Extracts and processes questions automatically  
- Generates a well-structured question paper  
- Reduces manual workload and improves efficiency  
- Ensures consistency in question paper format  

## Input
- One or more PDF files containing question banks  

## Output
- A finalized, structured question paper ready for use  

## Workflow
1. Upload or provide PDF question bank files  
2. Extract questions from the PDFs  
3. Organize questions based on predefined rules or structure  
4. Generate the final question paper  

## Technologies Used
- **Programming Language:** Python  
- **PDF Processing Libraries:** PyPDF2, pdfplumber, reportlab
